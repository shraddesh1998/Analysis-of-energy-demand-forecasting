🔋  Analysis-of-energy-demand-forecasting

This project focuses on forecasting energy demand in the German energy sector using Data Analytics, Machine Learning, and Business Intelligence tools. Historical energy demand, consumption, and weather datasets were analyzed to identify trends, seasonal patterns, and future demand behavior.

The project includes:
✔ Data Cleaning & Preprocessing  
✔ Exploratory Data Analysis (EDA)  
✔ Demand Forecasting using Machine Learning  
✔ Random Forest Forecasting Model  
✔ Time-Series Analysis  
✔ Interactive Power BI Dashboard  
✔ Weather Impact Analysis on Energy Demand  

Technologies Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI

Forecasting models such as Linear Regression, ARIMA, SARIMA, and Random Forest were implemented and compared. Random Forest achieved the best forecasting performance and was used to predict future energy demand for the next three months.

This project demonstrates how data analytical tools and machine learning techniques can support demand forecasting and decision-making in the energy sector.
